# Hangman Game 🎮

## 📌 Description
This is a simple console-based Hangman game built using Python. The player must guess a hidden word by entering letters or the full word within a limited number of attempts.

---

## ⚙️ Features
- Random word selection from a predefined list
- Guess by letter or full word
- Tracks already guessed letters
- Limited number of wrong attempts (6 chances)
- Win/Lose result display

---

## 🧠 Concepts Used
- Python loops (while loop)
- Conditional statements (if-elif-else)
- Lists and strings
- Random module
- User input handling
