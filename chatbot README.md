 Rule-Based Chatbot 🤖

 Description
This is a simple rule-based chatbot built using Python. It responds to user inputs using predefined rules and keywords.

 Features
- Responds to greetings (hello, hi, hey)
- Answers basic questions like "how are you"
- Provides simple help responses
- Exits when user types "bye"

Concepts Used
- if-elif statements
- loops (while loop)
- functions
- input/output handling

 Example Conversation
 You: hello
Bot: Hi!

You: how are you
Bot: I'm fine, thanks!

You: time
Bot: Current time is 14:35:10

You: bye
Bot: Goodbye! Have a nice day!

Project Type
Beginner-to-intermediate level Python project focused on logic building and rule-based AI simulation.
